Books API

GET /api/books

Retrieves a list of all books.

POST /api/books

Adds a new book to the library.

/api/books/:id

GET Retrieves a single book.

PUT Updates a single book.

DELETE Deletes a single book.

Example Request Body

POST /api/books

{ 
  "title": "The Great Gatsby", 
  "author": "F. Scott Fitzgerald", 
  "isbn": "978-0743273565", 
  "quantity": 5 
}

Users API

GET /api/users

Retrieves all registered users.

POST /api/users

Registers a new user.

/api/users/:id

GET Retrieves a single user.

DELETE Deletes a single user.

Borrow & Return API

POST /api/borrow

Borrows a book. Enforces business logic.

JSON BODY

{ "user_id": 1, "book_id": 12 }
POST /api/return

Returns a book. Updates the borrow log.

JSON BODY

{ "log_id": 7, "book_id": 12 }
